## 1. 产品简介

3C 结构件成本预评估工作台是一款基于 Python + Streamlit 的工程工具，用于帮助结构工程师、成本工程师和工艺评估人员快速完成消费电子结构件的早期成本估算。

工具支持 STEP 文件、Creo 截图、材料参数、工艺路线和成本规则的综合分析，并输出成本拆解、工程说明、优化建议和 PDF 报告。

当前网页地址：

```text
本地：http://localhost:9005/
云端：https://cost-workbench-3c-cy9pq3m3xhzczpihw7hhgc.streamlit.app/
```

## 2. 核心能力

当前产品包含以下能力：

- STEP 文件上传和基础识别。
- STEP 坐标点轻量外形尺寸解析。
- 本地安装几何内核后支持真实 STEP 几何解析。
- Creo 截图上传入口。

- 注塑件成本估算。
- CNC 铝件、钣金件、铝合金冲压、锌合金压铸、3D 打印快速估算。
- 材料库和材料参数联动。
- 工艺路线图展示。
- 成本拆解表。
- 优化建议。
- 页面报告预览。
- PDF 报告下载。
- Streamlit Community Cloud 免费部署。

## 3. 使用流程

### 3.1 第一步：输入分析

用户进入页面后，先选择一种输入方式：

- STEP 文件
- Creo 截图

STEP 文件方式：

1. 拖入或选择 `.stp/.step` 文件。
2. 系统读取文件名、文件大小、STEP schema 和实体数量。
3. 系统尝试解析几何尺寸。
4. 用户确认材料、工艺、重量、尺寸和单价。
5. 点击下一步。

Creo 截图方式：

1. 上传或拖入截图。
2. 系统根据文件名预识别视图类型。
3. 用户手动确认尺寸、材料和工艺参数。
4. 点击下一步。

第一页左侧下方提供咨询报价入口，展示收款码图片和备注：

```text
咨询更详细准确报价及工艺信息，请扫码，备注微信号
```

### 3.2 第二步：路线与成本

页面左侧显示工艺路线图，右侧显示成本拆解。

路线图内容包括：

- 工序名称
- 工序说明
- 成本项
- 风险提示
- 几何依据

成本拆解内容包括：

- 材料费
- 加工费
- 表面处理费
- 检验包装费
- 模具分摊费
- 良率调整
- 供应商管理利润率
- 最终单件预估价

### 3.3 第三步：优化建议

系统根据输入参数和工艺路线输出优化建议。

建议分类：

- 强度优化
- 成本优化
- 重量优化
- 制造风险

建议示例：

- 注塑件壁厚不均时，建议减薄厚胶位并使用筋位补强。
- CNC 件深腔较多时，建议减少深腔或放大内 R，降低小刀具加工时间。
- 钣金件折弯复杂时，建议优化折弯顺序并检查干涉。
- 冲压件拉伸深度较大时，提示开裂、回弹和整形成本风险。

## 4. STEP 解析说明

### 4.1 本地完整解析

本地安装 CadQuery/OCP 几何内核后，STEP 模块可读取：

- 最大外形长宽高
- 包围盒体积
- 真实实体体积
- 体积利用率
- 外表面积

安装方式：

```powershell
cd D:\work\codex\03-cost
.\install_geometry_kernel.ps1
```

安装后重启：

```powershell
cd D:\work\codex\03-cost
.\run_app.ps1
```

### 4.2 云端轻量解析

Streamlit Community Cloud 免费环境不直接安装 `cadquery/OCP`，否则容易出现 requirements 安装失败或构建超时。

云端当前采用轻量解析：

- 从 STEP 文本中的 `CARTESIAN_POINT` 估算外形长宽高。
- 计算包围盒体积。
- 估算包围盒外表面积。
- 不计算真实实体体积。

因此云端看到以下结果属于正常情况：

```text
status = 轻量解析成功
errorMessage = No module named 'cadquery'
volumeMm3 = None
volumeUtilization = None
```

含义：

- 文件基础信息读取成功。
- 坐标点外形估算成功。
- 真实 CAD 实体体积没有读取。
- 真实体积需要本地几何内核或 Docker 后端。

### 4.3 异常尺寸说明

如果轻量解析得到明显异常的尺寸，例如长度达到数十万毫米，通常说明 STEP 文件内存在全局坐标、参考点、装配点或非实体控制点。

此时应：

- 优先使用本地完整几何内核解析。
- 或手动输入长、宽、高和产品重量。
- 成本计算优先采用用户确认后的重量和尺寸。

## 5. 图片识别说明

当前图片模块是截图输入和基础预识别模块，不是完整 AI 视觉识别。

当前能力：

- 接收上传图片。
- 记录图片名称、大小和数量。
- 根据文件名判断视图类型。

文件名识别规则示例：

- `front`、`正视`、`主视`：前视图
- `top`、`俯视`：上视图
- `side`、`侧视`：侧视图
- `iso`、`3d`、`轴测`：3D 视图
- `section`、`剖视`、`detail`：剖视或局部细节

后续计划：

- OCR 读取截图中的尺寸文字。
- AI 识别筋位、卡扣、螺丝柱、折弯、孔位和倒角。
- 将图片识别结果用于成本参数、工艺风险和优化建议。

## 8. 成本计算说明

### 8.1 注塑件

注塑件采用行业常用公式：

```text
材料费 = 产品克重 / 1000 * 材料单价
加工费 = 注塑周期 * 机台秒费率 / 一模出数
模具分摊费 = 用户输入的单件摊销
良率调整后小计 = 基准小计 / (1 - 不良率)
最终单件预估价 = 良率调整后小计 * 1.15
```

内置机台费率：

- 90T = 0.016 元/秒
- 120T = 0.022 元/秒
- 160T = 0.028 元/秒

### 8.2 其他工艺

其他工艺采用规则估算：

```text
材料费 = 产品重量 * 材料单价 * 材料损耗系数
加工费 = 基础加工费 * 复杂度系数 * 批量系数 * 加工倍率 + 额外加工费
表面处理费 = 表面积 * 表面处理单价
检验包装费 = max(0.25, 加工费 * 0.06)
最终单件预估价 = 良率调整后小计 * 1.15
```

当前结果用于早期预估，不等同于供应商最终报价。

## 12. 使用注意事项

成本结果是预估值，需要工程师确认。

以下情况会降低结果可靠性：

- 没有真实 STEP 体积。
- 只有截图，没有尺寸标注。
- 材料单价不是企业实际采购价。
- 表面处理面积未确认。
- 批量、良率、模具摊销不准确。
- STEP 轻量解析得到异常长宽高。

推荐使用方式：

- 早期方案评审使用本工具快速判断成本方向。
- 关键项目使用本地几何内核读取真实体积。
- 最终报价仍需供应商结合图纸、工艺、模具和良率确认。
