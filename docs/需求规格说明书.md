# 3C 结构件成本预评估工作台需求规格说明书

版本：v1.0  
日期：2026-06-07  
项目位置：`D:\work\codex\03-cost`

## 1. 产品定位

本产品面向消费电子 3C 结构件的早期成本预评估，用于在方案评审、结构设计、供应商沟通和报价前快速完成输入分析、材料选型、工艺路线、成本拆解、报告生成和优化建议。

产品定位：

- 工程预评估工具
- 报价辅助工具
- 结构优化辅助工具
- 供应商沟通前的方案分析工具

产品不替代：

- Creo / CAD 原始模型
- 正式 2D 工程图
- 供应商最终报价
- 有限元仿真分析
- 正式制造审批

## 2. 目标用户

主要用户：

- 消费电子结构设计工程师
- 产品结构开发工程师
- 工艺工程师
- 成本工程师
- 报价预评估人员
- 供应商沟通和方案评审人员

典型场景：

- 早期结构方案成本评估
- 注塑、CNC、钣金、冲压、压铸、3D 打印件快速估算
- 材料和工艺路线对比
- STEP 零件几何分析
- Creo 截图辅助识别
- 强度、成本、重量优化建议生成
- PDF 报告输出和项目留档

## 3. 总体流程

产品采用三步式工作流：

1. 输入分析
   - 用户选择 STEP 文件或 Creo 截图作为输入方式。
   - 系统读取文件基础信息、几何参数、材料建议和工艺建议。
   - 用户确认或修正材料、重量、尺寸、单价和工艺参数。

2. 路线与成本
   - 系统根据工艺路线生成详细工序图。
   - 系统根据材料、重量、尺寸、表面积、批量、良率和工艺参数计算成本拆解。
   - 页面右侧展示材料费、加工费、表面处理费、模具分摊费、良率调整、供应商管理利润和最终单件预估价。

3. 优化建议
   - 系统结合输入数据、材料、工艺路线和成本结果输出优化建议。
   - 建议按强度、成本、重量和制造风险分类。
   - 支持生成完整报告，并支持 PDF 下载。

## 4. 输入方式需求

### 4.1 STEP 文件输入

支持拖入或选择：

- `.stp`
- `.step`

需要读取的基础信息：

- 文件名
- 文件大小
- STEP schema
- STEP entity count
- 解析状态
- 错误信息

完整几何内核可用时，应读取：

- 最大外形长 `lengthMm`
- 最大外形宽 `widthMm`
- 最大外形高 `heightMm`
- 包围盒体积 `bboxVolumeMm3`
- 真实实体体积 `volumeMm3`
- 体积利用率 `volumeUtilization`
- 外表面积 `outerSurfaceAreaMm2`

无完整几何内核时，应使用 STEP 文本坐标点轻量解析：

- 估算外形长宽高
- 估算包围盒体积
- 估算包围盒外表面积
- 明确提示真实实体体积为空
- 明确提示真实体积需要 CAD 几何内核

### 4.2 Creo 截图输入

支持上传或拖入图片文件。支持的图片可以来自：

- Creo 截图
- 截图工具导出的图片
- 本地图片文件

截图数量规则：

- 1-2 张截图：只允许草稿分析。
- 3 张及以上截图：可生成完整报告。

推荐截图组合：

- 1 张 3D 轴测或透视图
- 1 张主视图
- 1 张侧视图、俯视图、剖视图或局部细节图

当前阶段的图片识别能力：

- 接收图片名称、大小和数量。
- 根据文件名预识别视图类型。
- 支持后续扩展 OCR 和 AI 视觉识别。

后续图片识别目标：

- OCR 读取尺寸标注。
- 识别壁厚、孔径、R 角、折弯、筋位、卡扣、螺丝柱。
- 识别注塑倒扣、拔模角、厚胶位和缩水风险。
- 识别 CNC 深腔、小刀具、小 R、薄壁和去料率风险。
- 将图片识别结果纳入优化建议和成本参数确认。

## 5. 参数确认需求

第一页右侧为自动分析与参数确认区，不使用滑杆作为主要输入方式。所有关键参数应使用普通数字输入框、下拉框或可编辑单位输入。

需要支持的参数：

- 零件名称
- 推荐或选择工艺路线
- 材料选型
- 材料密度
- 材料参考单价
- 材料类别
- 材料或产品克重
- 重量单位
- 材料单价
- 单价单位
- 长、宽、高
- 尺寸单位
- 批量
- 复杂度
- 表面处理方式
- 表面积
- 供应商管理利润率
- 良率或不良率
- 损耗率

自动分析值必须允许用户修正。用户手动修改后的参数优先级高于系统自动识别值。

## 6. 材料选型需求

材料选型必须包含注塑件及其他常见 3C 材料/工艺，不得只覆盖金属件。

当前材料库覆盖：

- ABS
- PC
- PC+ABS
- PA66+GF
- POM
- AL6061
- AL5052
- SUS304
- SPCC
- Zamak 3
- SLA 树脂

材料数据字段：

- 材料类别
- 密度
- 参考单价
- 兼容工艺
- 强度说明
- 重量说明
- 成本说明
- 表面处理适配性
- 制造风险

材料选型联动规则：

- 选择注塑件时，应显示塑料材料。
- 选择 CNC 铝件时，应显示铝合金材料。
- 选择钣金件时，应显示钣金适用材料。
- 选择锌合金压铸时，应显示锌合金材料。
- 选择 3D 打印时，应显示原型材料。

## 7. 工艺路线需求

第二页左侧展示详细工艺路线图。路线图需要与当前工艺匹配，不同工艺显示不同工序和参数。

支持工艺：

- 注塑件
- CNC 铝件
- 铝合金冲压
- 钣金件
- 锌合金压铸
- 3D 打印

每个工序节点需要包含：

- 工序名称
- 工序说明
- 成本项
- 几何依据
- 风险提示

注塑件路线示例：

```text
材料确认 -> DFM/模流评估 -> 模具制造/试模 -> 注塑成型 -> 去披锋/二次加工 -> 表面处理 -> 检验包装
```

CNC 铝件路线示例：

```text
备料 -> 粗加工 -> 精加工 -> 钻孔/攻牙 -> 去毛刺 -> 表面处理 -> 检验包装
```

钣金件路线示例：

```text
开料 -> 冲孔/落料 -> 折弯 -> 压铆/攻牙 -> 焊接/点焊 -> 整形校正 -> 表面处理 -> 检验包装
```

铝合金冲压路线示例：

```text
卷料/板料备料 -> 开卷校平 -> 冲压落料 -> 冲孔/切边 -> 成型/拉伸 -> 整形校正 -> 去毛刺/清洗 -> 表面处理 -> 检验包装
```

## 8. 工艺参数需求

工艺参数必须随工艺路线变化，不能所有工艺都显示注塑参数。

注塑件参数：

- 注塑周期
- 机台吨位
- 一模出数
- 不良率
- 模具费摊销

内置注塑机台费率：

- 90T = 0.016 元/秒
- 120T = 0.022 元/秒
- 160T = 0.028 元/秒

CNC 参数：

- 批量
- 复杂度
- 加工费倍率
- 额外加工费
- 表面处理方式
- 表面积

钣金/冲压参数：

- 批量
- 复杂度
- 展开面积或表面积
- 表面处理方式
- 额外加工费

压铸参数：

- 批量
- 复杂度
- 表面处理方式
- 模具摊销或额外加工费

3D 打印参数：

- 批量
- 复杂度
- 表面处理方式
- 后处理费用

## 9. 成本计算需求

成本输出必须是分项拆解，不允许只输出总价。

### 9.1 注塑件行业公式

输入参数：

- 产品克重 g
- 材料每公斤单价 元/kg
- 注塑周期 s
- 机台吨位
- 一模出数
- 不良率
- 模具费摊销 元/件

计算公式：

```text
材料费 = 产品克重 / 1000 * 材料每公斤单价
加工费 = 注塑周期 * 机台秒费率 / 一模出数
模具分摊费 = 用户输入的单件摊销
良率调整后小计 = 基准小计 / (1 - 不良率)
最终单件预估价 = 良率调整后小计 * (1 + 15% 供应商管理利润率)
```

### 9.2 其他工艺估算

其他工艺需要根据材料费、基础加工费、复杂度、批量折扣、表面处理、检验包装、良率调整和供应商管理利润率计算。

成本项包括：

- 材料费
- 加工费
- 表面处理费
- 检验包装费
- 模具分摊费
- 不良率或良率调整
- 供应商管理利润率
- 最终单件预估价

供应商管理利润率默认值：

```text
15%
```

当关键参数缺失时，系统应提示“不可靠”或要求用户确认，不应给出假确定值。

## 10. 几何内核需求

本地环境需要支持真实 STEP 几何解析。目标能力：

- 读取真实实体体积
- 读取体积利用率
- 读取外表面积
- 读取精确包围盒
- 将体积与密度联动，自动反推产品重量

技术路线：

- 本地：Python + Streamlit + CadQuery/OCP。
- 免费云端：Streamlit Community Cloud + STEP 坐标点轻量解析。
- 完整云端：Docker 平台或云服务器部署 CAD 几何内核。

能力边界：

- Streamlit Community Cloud 免费环境不稳定适合安装 `cadquery/OCP`。
- 免费云端目前仅承诺轻量解析外形尺寸。
- 真实实体体积 `volumeMm3` 需要本地几何内核或 Docker 后端。

## 11. 页面结构需求

### 11.1 第一页：输入分析

左侧：

- 输入方式选择：STEP 文件或 Creo 截图。
- STEP 文件拖入/选择。
- 截图粘贴、拖入或本地上传。
- 两种输入方式任选其一。
- 左侧下方展示收款码咨询入口，位置必须避开所有输入框。
- 收款码图片下方显示备注：“咨询更详细准确报价及工艺信息，请扫码，备注微信号”。

右侧：

- 自动分析结果。
- 材料、重量、尺寸、单价和工艺参数确认。
- 所有参数可编辑。
- 点击“下一步”进入路线与成本。

### 11.2 第二页：路线与成本

左侧：

- 自动计算工艺路线图。
- 工序节点附图标或图示。
- 显示每个工序的说明、风险、成本项和几何依据。

右侧：

- 成本拆解表。
- 最终单件预估价。
- 工程说明。
- 报告预览。
- PDF 下载。

### 11.3 第三页：优化建议

内容：

- 强度优化建议。
- 成本优化建议。
- 重量优化建议。
- 制造风险建议。

建议必须结合实际输入参数，不输出空泛模板。

示例：

- 注塑件厚壁区域建议减薄并通过筋位补强。
- CNC 深腔结构建议减少深腔或放大内 R。
- 钣金件折弯过密时建议调整折弯顺序或增加避让。
- 冲压件拉伸深度过大时提示开裂和回弹风险。

- 最终单件预估价
- 工程说明

1. 第一部分：输入分析与成本

PDF 需要支持中文显示。

## 13. 部署需求

本项目采用 Python + Streamlit 技术路线。

部署目标分为三类：

### 13.1 本地运行

本地运行用于完整能力验证，尤其是 STEP 真实几何解析。

要求：

- 支持 Windows。
- 支持 `run_app.ps1` 或 `run_app.cmd` 启动。
- 默认访问 `http://localhost:9005/`。
- 支持安装几何内核后读取真实体积。

### 13.2 免费云端运行

免费云端使用 Streamlit Community Cloud。

要求：

- GitHub 仓库托管。
- 入口文件为 `streamlit_app.py`。
- Python 版本为 3.12。
- 不强制安装 `cadquery/OCP`。
- 使用轻量 STEP 解析避免云端构建失败。

### 13.3 完整云端运行

如果需要全国可访问并支持真实 STEP 体积解析，应使用支持 Docker 的平台：

- Zeabur
- Render
- 阿里云 ECS
- 其他支持 Docker 的云服务

完整云端应将 CAD 几何解析作为独立后端服务。

## 14. 非功能需求

易用性：

- 页面要清晰、美观。
- 使用三步式流程。
- 输入和结果并排展示。
- 数字、单位、材料参数要完整显示。

可靠性：

- 解析失败必须显示原因。
- 缺少几何内核不能误报为真实解析成功。
- 成本缺少关键参数时要提示风险。

隐私：

- 本地模式下文件留在本地。
- 云端上传 STEP 或截图时，应提示用户注意保密。

可维护性：

- 材料库、工艺路线、成本引擎、输入分析应模块化。
- 新材料、新工艺和新成本规则应可扩展。

## 16. 后续升级清单

优先级 P0：

- 本地几何内核稳定读取 STEP 真实体积。
- 用产品重量和密度反推材料体积，作为云端无几何内核时的成本兜底。
- 对异常外形尺寸增加校验，例如长度远大于常规 3C 零件时提示坐标点异常。

优先级 P1：

- OCR 读取截图尺寸标注。
- 图片结构特征识别。
- 优化建议绑定截图位置。
- 工艺参数随工艺进一步细化。

优先级 P2：

- Docker 云端几何解析后端。
- 国内云部署。
- 企业材料价格库。
- 多方案成本对比。
- 供应商报价导入与校准。
